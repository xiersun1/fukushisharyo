# AI完全自動化管理の運用案

このサイトは、WordPressを使わず、静的サイト + JSON + Decap CMS + AI監視で管理します。

| 役割 | 内容 |
| --- | --- |
| 静的サイト | 利用者が見る画面。HTML/CSS/JavaScriptだけで表示する |
| JSON | 応募先、受付状況、締切、公式URL、根拠メモを保存する |
| Decap CMS | 管理者がブラウザからJSONを編集する画面 |
| AI監視 | 公式ページやPDFを定期確認し、変更点を判定する |
| GitHub | 変更履歴を残す場所。誰が何を変えたか追える |
| 自動公開 | JSONや画面を更新したら自動でサイトに反映する |

## 自動監視の流れ

1. 公式ページ、PDF、申請フォームのURLを巡回する
2. 前回取得した内容と比較する
3. 「募集開始」「締切変更」「要項更新」「様式更新」「受付終了」を分類する
4. 重要な変更なら管理者に通知する
5. 問題がなければ `src/data/sources.json` を更新する
6. GitHubに変更を保存する
7. 静的サイトが自動公開される

## 無料で始める構成

| 項目 | 候補 |
| --- | --- |
| CMS | Decap CMS |
| ソース管理 | GitHub |
| 公開 | Cloudflare Pages |
| 自動実行 | GitHub Actions の無料枠 |
| 通知 | メール、RSS、LINE代替サービス、管理者ログ |
